# How to Load This Extension Locally

## Quick Steps

1. **Open Chrome Extensions Page**
   - Navigate to `chrome://extensions/` in your Chrome browser
   - Or go to: Chrome Menu → Extensions → Manage Extensions

2. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner

3. **Load the Extension**
   - Click the "Load unpacked" button
   - Select this directory: `/Users/anjalisharma/SocialMediaProfileLauncher`
   - Click "Select" or "Open"

4. **Verify Installation**
   - You should see "Profile Launcher" in your extensions list
   - The extension icon should appear in your Chrome toolbar

5. **Use the Extension**
   - Click the extension icon to open the popup
   - Click "Edit" to add your social media profiles
   - Save your links and start using the quick launcher!

## Troubleshooting

- If you see errors, check the browser console (right-click extension icon → Inspect popup)
- Make sure all files are in the same directory
- Ensure you're using Chrome (or Chromium-based browser like Edge, Brave)
